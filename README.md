# Dart Application Launcher
Basic application launcher for our internal applications. Uses Python to launch PS2Exe PowerShell scripts that install our in house applications.

# Install Instructions
- Create directory structure
    - C:\Program Files\Dart Applications\Dart Application Launcher\assets

- Copy files to where they need to be
    - PS2Exe installers -> C:\Program Files\Dart Applications\
    - Main exe -> C:\Program Files\Dart Applications\Dart Application Launcher\
    - Icons -> C:\Program Files\Dart Applications\Dart Application Launcher\assets
